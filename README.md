相片電影之協作拼貼系統 PhotoCollage
===
<img src="https://i.imgur.com/L9dL6NL.jpg" height="400" width="600">

# 主要功能
* 相片微電影：製作會說故事的相片回憶錄
* 相片拼貼：體驗高自由度的相片編輯
* 協作相簿：建立與好友共享的雲端相簿

# 系統特色
* 加入語音：可為相片錄製語音標籤，讓影片更生動
* 組合相片：可加入多張相片，自由編排版面
* 加入文字：可修改字型、顏色及文字呈現方式
* 更改背景：提供多款背景，也可將任一照片設為背景
* 分享相片：可透過 Facebook、Messenger、Line 分享成果
* 加入協作：新增相簿協作者，共同編輯雲端相簿
